# AI-based-hiring-prediction-system
AI-powered hiring prediction system that analyzes candidate profiles and predicts hiring outcomes using Machine Learning.


Overview:
The AI-Based Hiring Prediction System is a Machine Learning web application that predicts whether a candidate is likely to be hired based on resume-related information. The project demonstrates an end-to-end ML workflow, including data preprocessing, feature engineering, model training, evaluation, and deployment using Flask.

Features:
Candidate hiring prediction using Machine Learning
Resume data preprocessing
TF-IDF text feature extraction
Multiple ML model comparison
Flask-based web interface
Real-time prediction
